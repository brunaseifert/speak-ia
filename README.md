#  Speak-AI - Texto para Voz

Um simples aplicativo web que converte texto em áudio usando a **API de Síntese de Fala (SpeechSynthesis)** nativa do navegador.

## Sobre as funcionalidades

- Converte texto digitado em voz automaticamente.
- Permite escolher diferentes **vozes** disponíveis no navegador.
- Ajuste da **velocidade da fala**.
- Upload de arquivos `.txt` para leitura automática.
- Botão de **"Limpar / Parar"** que apaga o texto e interrompe a leitura.
- Interface simples, responsiva e intuitiva.
